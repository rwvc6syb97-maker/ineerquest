## 四、对外服务契约(供 report 已接入 / ai-chat T3-05 后续注入)

- `chat(req: ChatRequest): Promise<ChatResult>` — 一次性聚合结果,返回 `{ text, degraded, degradeReason?, model, traceId }`
- `chatStream(req: ChatRequest): AsyncGenerator<ChatStreamChunk>` — 逐 token 流式输出,始终产出可迭代流(超时/异常时 yield 兜底流,不抛出、不阻塞)
- `buildMessages(req)` — Prompt 分层编排(system / role / context / user)
- `checkRateLimit(callerId, scene)` — Redis 固定窗口限流,降级放行

> 已接入消费方:`apps/api/src/modules/report/report.service.ts`(`tryGenerateDeepText` 遍历付费段落调用 `llm.chat`,仅采纳 `!degraded && text.trim()` 文本,否则走 `DEEP_SECTION_FALLBACK` 兜底)。
